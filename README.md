# Simple Economic Growth Models

This repository contains implementations of simple economic growth models, like the Solow Growth Model.
The notebooks in this repo are intended only to help develop intuition about the dynamics of these models, and are not meant to be rigorous or comprehensive.

In particular, I have tried to model the impact of automation (for example, through the use LLMs) on economic growth, and the implications of this for the future of work and the economy.
